# 真太阳时功能说明

## 功能介绍

真太阳时是基于太阳实际位置计算的时间，与我们日常使用的标准时间（平太阳时）不同。在传统命理学中，真太阳时比标准时间更精确，特别是在计算生辰八字时。

## 为什么需要真太阳时？

1. **天文精度**：标准时间是人为划分的时区时间，而真太阳时反映太阳的实际位置
2. **传统要求**：古代命理学使用的就是真太阳时
3. **精确计算**：特别是在时辰边界附近，真太阳时校正可能会影响八字结果

## 功能特点

### 🌍 地理位置支持
- 内置60+个中国主要城市的经纬度数据
- 支持手动输入精确经度
- 覆盖全国各大省会和重要城市

### ⚡ 计算方法
- **简化计算**：速度快，精度足够日常使用
- **精确计算**：更高精度，适用于专业场景

### 🎯 实时预览
- 显示标准时间与真太阳时的差异
- 实时显示使用的经度信息
- 直观的时间对比

## 使用方法

### 1. 启用真太阳时
在八字计算页面的"真太阳时设置"卡片中，打开真太阳时开关。

### 2. 设置位置
**方式一：选择城市**
- 从下拉菜单中选择出生地城市
- 系统自动使用该城市的经度

**方式二：输入经度**
- 在经度输入框中输入精确的经度值
- 东经为正数，西经为负数
- 例如：北京 116.4074°

### 3. 选择计算方法
- **简化计算（推荐）**：适合大多数情况
- **精确计算**：天文级精度，计算稍慢

### 4. 查看预览
系统会实时显示：
- 标准时间
- 真太阳时
- 时间差异
- 使用经度

### 5. 执行计算
点击"计算八字命理"按钮，结果会显示时间校正信息。

## 技术原理

### 时间校正公式
```
真太阳时 = 标准时间 + 经度校正 + 均时差
```

### 经度校正
```
经度校正 = (当地经度 - 标准时区中央经线) × 4分钟/度
```

### 均时差
考虑地球椭圆轨道和地轴倾斜的影响，最大可达±16分钟。

## 注意事项

1. **时辰边界**：在子时（23:00-1:00）等时辰边界时，时间校正可能影响八字结果
2. **地理位置**：请选择准确的出生地，误差会影响计算精度
3. **历史考证**：古代使用地方平太阳时，现代真太阳时校正更为精确
4. **计算耗时**：精确计算模式稍慢，简化模式已足够准确

## 兼容性说明

- ✅ 完全向后兼容：不启用真太阳时时，计算结果与之前完全一致
- ✅ 可选功能：用户可以自由选择是否启用
- ✅ 实时对比：可以同时看到校正前后的时间信息

## 使用建议

1. **一般用户**：建议使用简化计算模式，选择最近的城市
2. **专业用户**：可以使用精确经度和精确计算模式
3. **学习对比**：可以尝试对比启用/不启用真太阳时的差异
